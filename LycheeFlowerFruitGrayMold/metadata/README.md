# 气候数据处理工具

## 概述
这是一个用于处理气象数据的Python脚本，能够从Excel文件中读取每日气候数据，并计算3天、5天、7天和10天的滑动窗口统计值。

## 主要功能
- 读取Excel格式的气候数据
- 计算多时间窗口的平均值和累积值
- 自动处理数据不足的情况（如开始的几天数据）
- 输出Excel和CSV两种格式

## 文件说明
- `process_climate_data.py` - 主处理脚本
- `使用说明.md` - 详细的使用说明文档
- `G1099.xlsx` - 输入数据文件（需要用户提供）
- `影响因子1103.xlsx` - 输出Excel文件
- `影响因子1103.csv` - 输出CSV文件

## 快速使用
```bash
# 安装依赖
pip install pandas openpyxl numpy

# 运行脚本
python process_climate_data.py
```

## 输入数据格式
Excel文件应包含6列数据（按顺序）：
1. 日期
2. 平均气温
3. 雨量
4. 相对湿度
5. 降雨时数
6. 日照时数

## 输出数据格式
为每个时间窗口（3、5、7、10天）计算：
- 平均气温
- 累积雨量
- 平均相对湿度
- 累积降雨时数
- 累积日照时数

共生成21列输出数据（1列日期 + 4个窗口×5个指标）。

## 技术特点
- 使用pandas进行高效数据处理
- 自动类型转换和缺失值处理
- 详细的进度和统计信息输出
- 支持中文列名和数据
- 容错处理，确保数据不足时也能正常计算
